JSEL デモ

使用準備

HTMLファイルのどこにでもロードする事が出来ます


      <script src="path/js/Jsel.js"></script>

JSONファイルの形式

下記のような形式のJSONファイルを用意してください

{
        "PARENT_KEY1": {
            "name": "PARENT_NAME1",
            "child": {
                "CHILD_KEY1": {
                    "name": "CHILD_NAME1"
                }, 
                "CHILD_KEY2": {
                    "name": "CHILD_NAME2"
                }
            } 
        }, 
        "PARENT_KEY2": {
            "name": "PARENT_NAME2",
            "child": {
                "CHILD_KEY3": {
                    "name": "CHILD_NAME3"
                }, 
                "CHILD_KEY4": {
                    "name": "CHILD_NAME4"
                }
            }
        },
        .
        .
        .
        .
    }

使用例

ノーマルな使い方


      <script name="Jsel" data-path="your/json/filepath" data-pid="parent" data-cid="child"></script>

空項目と空項目の表示メッセージを設定する


      <script name="Jsel" data-path="your/json/filepath" data-pid="parent2" data-cid="child2" data-empty="true" data-text="なし"></script>

デフォルト値を設定する


      <script name="Jsel" data-path="your/json/filepath" data-pid="parent3" data-cid="child3" data-pval="13" data-cval="13120"></script>
